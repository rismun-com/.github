# ریسمان

ریسمان یک شرکت نرم‌افزاری ایرانی (تأسیس ۱۳۸۸) است که پلتفرم‌های دیجیتال مأموریت‌محور برای سازمان‌ها و بخش عمومی می‌سازد. تخصص ما معماری‌های مایکروسرویس و مایکروفِرِنت‌اند، سیستم‌های API‑First و جریان‌های امنِ داده در محیط‌های درون‌سازمانی یا ابر خصوصی/هیبرید است. ترکیب تفکر محصول با مهندسی دقیق، خروجی قابل‌اعتماد، قابل‌مشاهده و حریم‌خصوصی‌محور در مقیاس را تضمین می‌کند.

## نمای کلی شرکت
- نام فارسی: پردازش اطلاعات ریسمان
- حوزه‌ها: دولت الکترونیک، اتوماسیون سازمانی، سامانه‌های مالی و اداری
- تمایزها: معماری استانداردمحور، انضباط تحویل، چندزبانه (FA/EN/AR) و پشتیبانی کامل RTL

## چه می‌کنیم
- مهندسی پلتفرم
  - شتاب‌دهنده‌های «سکوی دیجیتال ریسمان»: هویت و دسترسی، جریان‌کار/BPMS، فرم‌ها، اسناد، اعلان‌ها، هاب یکپارچه‌سازی.
- توسعه محصول
  - از اکتشاف تا UX/UI، پیاده‌سازی، استقرار و عملیات؛ با خروجی‌های قابل‌اندازه‌گیری (SLI/SLO) و چرخه انتشار روشن.
- مایکروسرویس و مایکروفِرِنت‌اند
  - طراحی دامنه‌محور (DDD)، رویدادمحور، درگاه API و سرویس‌مش؛ مقیاس‌پذیری و انتشار مستقل.
- یکپارچه‌سازی و تعامل‌پذیری
  - REST/GraphQL/gRPC، پیام‌رسانی، وبهوک‌ها، و CDC؛ کانکتورهای ERP/CRM/Tax و سامانه‌های دولتی.
- فرایند و اتوماسیون
  - مدل‌سازی BPMN، SLA و ارجاع، امضا الکترونیک، ثبت وقایع و انطباق‌پذیری.
- داده و توانمندسازی هوش مصنوعی
  - جریان‌های داده، جستجو/تحلیل، خروجی‌های BI؛ دریافت هوشمند و جستجوی مبتنی بر بازیابی (RAG) در صورت نیاز.
- ابر و DevOps
  - کانتینر، CI/CD، IaC، عملیات Kubernetes؛ تحویل تدریجی و برابری محیط‌ها.
- امنیت و حکمرانی
  - OIDC، RBAC/ABAC، رمزنگاری حین انتقال/در سکون، ثبت ممیزی؛ حریم‌خصوصی‌محور و حداقل‌دسترسی.
- آموزش و پشتیبانی
  - توانمندسازی، مستندسازی و نگهداری بلندمدت برای موفقیت در بهره‌برداری.

## محصولات و راهکارهای شاخص
- سکوی دیجیتال ریسمان (RDP)
  - پایه‌ی یکپارچه مایکروسرویس‌ها و مایکروفِرِنت‌اندها: هویت، جریان‌کار، فرم‌ها، اسناد، اعلان‌ها، ثبت وقایع، و هاب یکپارچه‌سازی.
  - فناوری: مایکروفِرِنت‌اندها (Next.js، Module Federation)، مایکروسرویس‌ها (.NET، Node.js/NestJS)، gRPC داخلی، REST/GraphQL بیرونی، Kafka/RabbitMQ، PostgreSQL/MongoDB/Redis، OpenSearch، ذخیره‌ساز سازگار با S3، درگاه API، سرویس‌مش، OIDC، OpenTelemetry.

- RISAPP (Low‑Code فرم و فرآیند)
  - طراح WYSIWYG، فرم‌های مبتنی بر طرح‌واره، موتور قواعد، گردش‌کار چندمرحله‌ای، نسخه‌بندی، Feature Flag، PWA و آماده‌ی موبایل.
  - فناوری: مایکروفِرِنت‌اندهای Next.js، بک‌اند Node.js/NestJS یا .NET، درگاه GraphQL، gRPC بین‌سرویسی، JSON Schema، PostgreSQL + MongoDB، Redis.

- اتوماسیون اداری و جریان‌کار (BPMS)
  - مدل‌سازی BPMN، مکاتبات، ارجاع وظایف، SLA و Escalation، امضای دیجیتال/PKI/HSM، مدیریت اسناد و ثبت کامل وقایع.
  - فناوری: سرویس‌های .NET، مایکروفِرِنت‌اندها، gRPC، رویدادهای Kafka، جستجوی متنی OpenSearch، مخزن اسناد سازگار با S3.

- سامانه مدیریت منابع انسانی (HRMS)
  - اطلاعات پرسنل، جذب و ورود، هم‌ترازی حقوق، مرخصی/حضور، عملکرد، سلف‌سرویس؛ چندمستاجری، بومی‌سازی و کنترل دسترسی قوی.
  - فناوری: مایکروسرویس‌های .NET، مایکروفِرِنت‌اندهای Next.js، PostgreSQL، Redis، RabbitMQ؛ OIDC؛ API مبتنی بر GraphQL.

- مدیریت اسناد و بایگانی الکترونیکی (EDRMS)
  - دریافت، OCR/طبقه‌بندی، نگهداشت و قفل‌های حقوقی، نسخه‌بندی، کنترل دسترسی، همکاری، پشتیبان‌گیری امن، اشتراک‌گذاری Zero‑Trust؛ جستجوی برداری.
  - فناوری: OpenSearch/Elasticsearch، ذخیره‌ساز سازگار با S3، پایپ‌لاین‌های ایندکس Kafka، سرویس‌های .NET/Node.js، مایکروفِرِنت‌اندهای Next.js.

- پیام‌رسانی و اعلان همه‌کاناله
  - SMS، ایمیل، پوش و درون‌برنامه؛ الگوها و بومی‌سازی؛ Rate Limiting، گزارش تحویل، Failover چندسرویس‌دهنده؛ وبهوک و رویداد.
  - فناوری: سرویس‌های Node.js، Kafka/RabbitMQ، وبهوک‌های REST، OIDC، OpenTelemetry.

- هاب یکپارچه‌سازی و درگاه API
  - کانکتورهای ERP/CRM/Tax؛ APIهای REST/GraphQL؛ پل gRPC؛ CDC با Debezium؛ تبدیل و زمان‌بندی؛ مدیریت اسرار؛ سرویس‌مش.
  - فناوری: درگاه API، فدراسیون GraphQL، پروکسی gRPC، Kafka، PostgreSQL، Vault، Istio/Linkerd.

- بنیاد داده و تحلیل
  - گزارش‌دهی عملیاتی، پایپ‌لاین‌های شبه‌بلادرنگ، نماهای مادی‌سازی‌شده، خروجی‌های BI، کاتالوگ و تبار داده؛ CDC از سامانه‌های مبدا.
  - فناوری: Kafka Streams، PostgreSQL، OpenSearch، ذخیره‌سازی ابجکتی.

- ماژول‌های اختیاری و R&D (فرضی)
  - فرم‌ها و استخراج سند با کمک هوش مصنوعی، جستجوی RAG روی بایگانی‌ها، کشف ناهنجاری رویدادهای مالی، هلپ‌دسک محاوره‌ای.
  - فناوری: پایگاه‌های داده برداری، پایپ‌لاین‌های تع嵌، مدل‌های On‑Prem/ابر خصوصی، میان‌افزار Guardrail، حریم‌خصوصی‌محور.

## پروژه‌های منتخب
- اپ «بیستی» — پیام، رویداد، تیکت، مکاتبات و پرداخت قبوض.
  - تکنولوژی: React Native + پورتال وب (Next.js)، مایکروسرویس‌های NestJS (Node.js)، gRPC داخلی + REST/GraphQL بیرونی، PostgreSQL، Redis؛ Docker/Kubernetes؛ OIDC؛ OpenTelemetry.

- «دکتر سازه» — بازرسی ساختمان و خدمات کارشناسی.
  - تکنولوژی: Next.js (React + TypeScript) به‌صورت مایکروفِرِنت‌اند، مایکروسرویس‌های NestJS، gRPC + درگاه REST، PostgreSQL، OpenSearch، ذخیره‌ساز سازگار با S3؛ CI/CD با GitHub Actions.

- فرم‌ساز شعب بانک ملت (RISAPP) — گردش‌کار بانکی و طراحی Low‑Code.
  - تکنولوژی: مایکروفِرِنت‌اندها (Next.js)، مایکروسرویس‌های .NET + gRPC، درگاه REST/OpenAPI، Kafka، MongoDB (تعاریف فرم)، PostgreSQL (تراکنشی)، Redis؛ استقرار Kubernetes با Helm/Kustomize.

- منابع انسانی — پست ایران
  - تکنولوژی: مایکروسرویس‌های .NET (Minimal APIs)، gRPC داخلی، مایکروفِرِنت‌اندهای Next.js، PostgreSQL، RabbitMQ، RBAC/ABAC با Keycloak (OIDC)، داشبورد با Grafana.

- اسناد حسابداری — TCEO
  - تکنولوژی: سرویس‌های .NET، پردازش رویدادمحور (Kafka)، PostgreSQL، خروجی‌های OLAP/BI، ثبت ممیزی و لاگ‌های تغییرناپذیر؛ رصد با OpenTelemetry + Prometheus/Grafana.

- اتوماسیون اداری — راه‌آهن ج.ا.ا.
  - تکنولوژی: مایکروفِرِنت‌اندها (Next.js)، مایکروسرویس‌های .NET، gRPC بین‌سرویسی، Elasticsearch جستجوی متن/سند، مخزن اسناد سازگار با S3، Keycloak برای SSO/OIDC؛ استقرار چندمحیطی روی Kubernetes.

## شیوه‌های مهندسی
- معماری: سرویس‌های ماژولار، مرزبندی شفاف، API‑First؛ فرانت‌اندهای SPA با قراردادهای تایپ‌شده.
- کیفیت: آزمون خودکار، بازبینی کد، تحلیل ایستا و CI برای بیلدهای تکرارپذیر.
- تحویل: انتشار تدریجی، همسانی محیط‌ها، استقرار قابل‌مشاهده.
- امنیت: RBAC/ABAC، حداقل‌دسترسی، ثبت رویداد و کدنویسی ایمن.
- مشاهده‌پذیری: لاگ ساختاریافته و متریک‌های عملیاتی.

## ماتریس فناوری
- ترجیحی (پیاده‌سازی‌های جدید)
  - Backend: .NET (ASP.NET Core), C#; Node.js (NestJS/Express)
  - APIs: REST (OpenAPI), GraphQL, gRPC, WebSocket
  - Frontend: React + TypeScript; Next.js (App Router), Vite
  - UI: Tailwind CSS, Material UI; Design Tokens, Storybook
  - State/Data: React Query/RTK Query; React Hook Form; Zod
  - Mobile: React Native (TypeScript)
  - Data: PostgreSQL, SQL Server, MongoDB; Redis
  - Search/Analytics: OpenSearch/Elasticsearch; BI Exports
  - Messaging: Kafka (events), RabbitMQ (queues)
  - AuthN/Z: OIDC/OAuth2, JWT, RBAC/ABAC
  - DevOps: Docker, Kubernetes, GitHub Actions; Helm/Kustomize; Terraform (IaC)
  - Observability: OpenTelemetry, Prometheus, Grafana, ELK/OpenSearch

- پشتیبانی‌شده (سامانه‌های موجود)
  - ASP.NET (Full Framework), HTML/CSS/Bootstrap
  - Node.js + React, Material UI
  - SQL Server; REST Web Services; SMS Gateways
